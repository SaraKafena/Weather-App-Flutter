# Weather App

A Flutter-based weather application that retrieves real-time weather information using a weather API. Users can search for any city and instantly view current temperature and weather conditions.

## Features

- Search weather by city name
- Real-time weather data
- Display current temperature
- Display weather condition
- HTTP API integration
- JSON data parsing
- Asynchronous data handling
- Clean and responsive UI

## Technologies Used

- Flutter
- Dart
- REST API
- HTTP Package
- JSON Parsing
- Async Programming (Future & async/await)

## How It Works

1. Enter a city name.
2. The app sends an HTTP request to the weather API.
3. The API returns weather data in JSON format.
4. The app parses the response and displays:
   - Temperature
   - Weather Condition
